##### 필요소스

- 룰렛 이미지 (확장자: png)
- 핀 이미지

## 이미지 준비

- 이미지 이름은 **이미지이름-이미지번호.png**
  ex) rouletteImg-2.png , 여기서 이미지 번호는 룰렛의 칸 수를 나타냄.
- 이미지 모양
